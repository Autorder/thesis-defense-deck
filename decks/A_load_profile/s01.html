How the load profile is built

?
Examiner question

How exactly is the load profile built — data source, hour allocation, weekends and holidays — and how do the same rules apply to a hot city (Beer Sheva) versus a mild one (Haifa)?

In brief

A deterministic, temperature-driven profile that reproduces a household's real behaviour — seasonal, intra-day and per-climate. There is no single “true” profile: the model captures the main physical drivers transparently, applies the same rules to every city, and closes exactly to the annual budget.
