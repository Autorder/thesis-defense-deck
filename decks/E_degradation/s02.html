Battery initialization & hourly update

// end-of-year target
Q₁ = min( Q₀ , target )
// hourly fade step (linear in-year)
ΔQ = ( Q₀ − Q₁ ) / 8760
Q_new = max( 0 , Q − ΔQ )
// power scales with capacity
P_max = P₀ · ( Q / Q₀ )^ηP
// SOC bounds contract with Q
SOC ∈ [ f_min·Q , f_max·Q ]
// end of life
EOL : Q_new ≤ eol_frac · Q₀
Linear in-year, geometric across years

Inside a year, a constant linear step; the year-to-year targets follow a constant-fraction curve (60/70/80% at yr 9) → a convex decade, not a straight line.

Power & SOC track capacity

Max power drops by a power law P₀·(Q/Q₀)^ηP, and the SOC window contracts together with the current capacity.

Counted, not used

Equivalent full cycles (EFC) are recorded but don't drive the fade; temperature is retained but never enters the fade law.

The fade is exogenous and transparent by design — a warranty-style retention target, deliberately chosen over an uncalibrated cell-aging curve.
