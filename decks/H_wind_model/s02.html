The wind model and the power curve

1 kW VAWT, from TMY wind

Power = curve × hourly TMY wind speed × turbine count — computed for all 8,760 hours.

The curve — 3 points, linear

Cut-in 3 m/s; 0.05 → 0.4 → 1.0 kW at 3 / 7 / 11 m/s; rated 1 kW at 11, flat above (no cut-out).

Hub height not modeled

The TMY wind speed is used directly, with no height adjustment — a declared simplification.

The same 3-point curve drives every run — the city difference comes entirely from local wind speed, not from a different turbine.
